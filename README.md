# Battery Widget

> Only Support macOS

![Screen shot](https://raw.githubusercontent.com/m00p1ng/battery-widget/master/images/screenshot.png)

**Clone this repository**

```
$ git clone https://github.com/m00p1ng/battery-widget
```

**Install Dependency pakages**
```
$ yarn
or 
$ npm install
```

**Build**
```
$ yarn build
or
$ npm run build
```
